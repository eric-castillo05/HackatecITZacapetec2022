
# HackatecITZacapetec2022

This project is an application of multiple linear regression. The project was created for the [HackatecITZacapetec2022] hackathon.

## Technologies
- Python 3.6 or higher
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
